# C'Propre - Pressing Professionnel

Site web professionnel pour service de pressing avec collecte à domicile.

## ✨ Fonctionnalités

- ✅ **Design professionnel** moderne style 5àsec
- ✅ **12+ images professionnelles** haute qualité (Unsplash)
- ✅ **3 formules d'abonnement** (15L, 30L, 50L)
- ✅ **Collecte tous les lundis** automatique
- ✅ **Système de suivi** par code/QR code en temps réel
- ✅ **11 sections complètes** : Hero, Services, Features, Gallery, Subscriptions, How It Works, Testimonials, CTA, Tracking, Footer
- ✅ **100% Responsive** design mobile/tablet/desktop
- ✅ **Performance optimisée** Next.js 15 + Turbopack

## Technologies

- Next.js 15
- TypeScript
- Tailwind CSS
- Lucide Icons

## Installation

```bash
npm install
```

## Démarrage

```bash
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000)
